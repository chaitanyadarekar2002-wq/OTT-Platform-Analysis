# 🎬 OTT Platform Content Analysis Dashboard

## 📌 Project Overview
The entertainment industry has rapidly shifted toward digital streaming platforms such as Netflix, Amazon Prime Video, and Disney+.  
This project analyzes and compares content available across these platforms using **Python** and **Power BI** to uncover trends, audience preferences, and platform strategies.

The dashboard provides interactive insights into:
- Content distribution
- Genre popularity
- Ratings analysis
- Release trends
- Platform comparison
- Country-wise content availability

---

# 🚀 Objectives
- Analyze OTT platform content libraries
- Compare Movies vs TV Shows across platforms
- Identify top genres and audience ratings
- Discover content growth trends over the years
- Build an interactive Power BI dashboard for business insights

# 🔍 Key Insights

- Netflix contains the largest content library among all platforms.
- Drama and Comedy are the most dominant genres.
- Movies significantly outnumber TV Shows across platforms.
- Content production increased rapidly after 2015.
- Disney+ focuses more on family-oriented content.
